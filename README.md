**HOW TO SETUP**
1. Press **Win + R** , write **"cmd"**, and hit enter
2. Navigate to your project folder
   use the **cd** command, For example: cd C:\Users\YourName\Desktop\project-folder
3. Run the the install command
   **npm install**
4. Click **start.bat**
